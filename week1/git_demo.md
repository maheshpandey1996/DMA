### One-time global configuration settings.

```bash
1. git config --global user.name "binay-shah"
```

### Initializing the repo

```bash
mkdir repo
cd repo
git init repo
ls -a
```

### Commit
```bash
touch file.txt
git status
git add .
git status
git commit -m "Initial commit"
git log
```
